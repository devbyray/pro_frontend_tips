Do you use Git for your private development and for the company you work for, but want to use for example Github and Bitbucket with different accounts on 1 machine? That’s really easy to setup!

There are 2 ways you can arrange it. 1 way is to use the command prompt, the other way is with Sourcetree. In my explanation i will assume that you allready installed Git and/or Sourcetree and that your allready a bit familiar with how Git works.

“The Command Prompt” way of changing the username and email

If you use Git with the command prompt and want to change the username and email for an specific repository, than use this command:

git config user.name "Username"`
git config user.email 'name@yourdomain.com"

So from now on, if you commit any change to your repo this username and email will be used. So easy is it!

“The SourceTree” way of changing the username and email

For those users who use SourceTree it is also that easy! I assume you allready cloned an repo to your machine with SourceTree. If not just do that how you allways do that.

Select the settings of the repository

Disable the checkbox and fill in your username and email

Automatic user selection based on domain name

So that’s it! I know there is an way to setup accounts connected to domain name of your Git repository. But haven’t that worked yet, so i will search for that. But this article can help you to configure that.
